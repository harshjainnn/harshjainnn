## 🧠 About Me

I'm a final-year **Computer Science** student at **Vellore Institute of Technology (CGPA 8.57)**, currently working as a **Software Developer at Ximble**, a logistics platform, where I build an **AI-driven QA agent system** — ingesting GitHub webhooks, running multi-repo impact analysis with Gemini, generating HTTP test plans from Arazzo specs, and executing them autonomously against staging environments.
